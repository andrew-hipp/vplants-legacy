Scientific Name Checkist

The vPlants Scientific Name Checklist contains plant names (and their synonymy) for the Chicago Region. This checklist is an intersection of the following data sources:

  • USDA PLANTS National Database (http://plants.usda.gov/)
  • Plants of the Chicago Region (Swink & Wilhelm)

Family Name Index

Scientific Names listed by Family - presented in HTML, CSV, or XML format.

Technical Sidenote: This listing is created "on the fly" from an Oracle database using XML. This is the same architecture we will utilize on the vPlants Portal when publishing data.
Checklist Additions

This is a listing of names we have added to the checklist. These names were, for various reasons, not included in the USDA data set. The Name Codes for these additions all begin with "VP-".

These names are included in the Family Name Index. I've listed them here separately only for reference.
