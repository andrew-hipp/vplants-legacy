Enable Java(TM) Web Services Developer Pack Callback to receive updated information related to Sun's Web services effort.
(Select Decline to disable)

Sun Microsystems, Inc. respects your desire for privacy.
By enabling Java WSDP Callback, you will be returning information about your usage of Java WSDP to Sun. Only information corresponding to your usage of Java WSDP will be collected and returned to Sun by the Callback Functionality. This information will be used to better serve your needs through Java WSDP and to provide you with information about the latest updates related to Sun. No personal information will be collected via the Callback Functionality.

The Java WSDP Callback collects a limited amount of non-personal information from your computer and sends it to Sun. This includes the Web container which Java WSDP was integrated on, the technologies chosen during installation, and the version of Java WSDP. You can disable sending this information by declining to accept this policy initially or by using the Preferences options to never contact the server at any time.
